Apple Store (Example) – Assignment 3 Landing Page
====================================================

Files
- index.html
- style.css
- /assets (local images)

How to run
1) Open index.html in any browser (Chrome/Edge/Firefox).
2) Internet is recommended because Bootstrap and Bootstrap Icons are loaded via CDN.
   If your lecturer requires fully offline, you can download Bootstrap and link locally.

Features (Bootstrap)
- Responsive navbar + collapse
- Carousel (hero)
- Product cards in responsive grid
- Offcanvas cart + toast notifications
- Modal promo
- Accordion FAQ
- Search + sort products (JS)
